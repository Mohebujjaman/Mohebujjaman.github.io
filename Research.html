Magnetic field magnitude at T=40 for a MHD channel flow over a step.


My research interests lie in finite element analysis and large scale simulations of partial differential equations, which include Newtonian Navier-Stokes equations, Magnetohydrodynamics, fast algorithms, large scale pararrel implementation of fluid flow problems using deal.II(massively parallel). Currently I am working with Dr. Leo Rebholz and Dr Traian Iliescu on "Reduced order modeling for 2D flow past a cylinder". For my research, I write code in Dealii, a C++ finite element open source library, matlab, FreeFem++, R, SAS, Python, and FORTRAN.
